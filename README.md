# HR Analytics Dashboard (Power BI)

## 📌 Project Overview

This project is an HR Analytics Dashboard created using Power BI.

The dashboard helps HR teams analyze employee attendance, Work From Home (WFH), and Sick Leave (SL). It provides interactive visualizations that make it easy to understand attendance patterns and monitor employee performance.

---

## 🚀 Features

- Employee Attendance Percentage
- Work From Home (WFH) Percentage
- Sick Leave (SL) Percentage
- Monthly Attendance Filter
- Attendance Trend Analysis
- Employee-wise Attendance Report
- Day-wise Attendance Analysis
- Interactive Dashboard

---

## 🛠️ Tools Used

- Power BI
- Power Query
- DAX
- Data Visualization

---

## 📊 Dashboard Insights

The dashboard includes:

- Overall Attendance %
- Overall WFH %
- Overall Sick Leave %
- Attendance trend over time
- Employee attendance summary
- Day-wise attendance analysis

---

## 🎯 Project Objective

The main goal of this project is to help HR departments monitor employee attendance, identify attendance trends, and make better decisions using data.
